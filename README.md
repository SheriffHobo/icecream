# Ice Cream Eater
Ice Cream Eater is a restaurant app that lets users input the names of Ice Creams they'd like to eat. Then click on the added Ice Cream to Devour it. Once Devoured, they can click on that same falvor to remove it from the list.

## Technologies Used
* Node
* Express
* Handlebars
* Bootstrap
* MySQL
* JawsDB

## Hosted on Heroku
Follow this [link](https://ice-cream-eater.herokuapp.com/ "Ice Cream Eater on Heroku") to view the web app.
